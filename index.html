VIA YOLO ZIP Export

This is an alternative implementation of the YOLO export functionality for the VGG Image Annotator (VIA). This implementation provides a ZIP-based export option that creates individual text files for each image's annotations.

Features

  • Exports annotations in YOLO format (normalized coordinates)
  • Creates a ZIP file containing:
    • classes.txt with class names
    • Individual .txt files for each image's annotations
  • Compatible with YOLOv8 training
  • No additional post-processing required

Download

Download the ZIP-based YOLO export implementation:

Download yolo_zip_export.js

Usage

1. Include the following files in your VIA project:

<script src="https://cdnjs.cloudflare.com/ajax/libs/jszip/3.10.1/jszip.min.js"></script>
<script src="yolo_zip_export.js"></script>

2. Add the export option to your VIA interface:

{
  name: 'Export as YOLO (ZIP)',
  action: function() {
    this.export_to_yolo_zip();
  }
}

Example Output

The export will create a ZIP file containing:

yolo_annotations.zip
├── classes.txt
├── image1.txt
├── image2.txt
└── ...
